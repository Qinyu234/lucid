PROJECT TODO
Graph Compiler Architecture
Source of truth: graph/ + templates/ + rules/
Target: self-hosting system that can rewrite itself via graph

设计原则（不随 phase 变化）

graph 是 L1 source of truth，code 是 compiler artifact
tree（语义意图）和 graph（执行结构）双向影响，可回溯，可异步
所有模块满足 fn(inputs, meta) -> dict 签名（自举前提）
schema 字段只能通过 compiler/accessor/schema_accessor.py 访问
没有 any 类型，union 必须显式列出 variants
没有通过测试的 template 不能注册进 registry
core（loader/validator/executor/codegen）零 web 框架依赖

## TASK-002
状态：已完成
做什么：定义 Canonical IR schema
输入：project_direction_outline.md 第五节
输出：project/compiler/ir/schema.py

要求：
  IR 是结构契约，不是语义 IR。
  IR 只表达可静态展开的执行结构。

  定义 ir_node dict：
  ```python
  {
    "id": str,
    "kind": str,        # function / call / assign / branch / loop / return / side_effect / opaque
    "label": str,
    "scope": str,       # 所属 scope id
    "inputs": list,     # 依赖的 ir_node id 列表
    "outputs": list,    # 产出的 ir_node id 列表
    "meta": dict,       # 附加信息（side_effect flag，safe_level 等）
    "source_ref": dict, # {"path": str, "lineno": int, "end_lineno": int}
  }
  ```

  定义 ir_scope dict：
  ```python
  {
    "id": str,
    "kind": str,   # module / function / class / block
    "parent": str | None,
    "node_ids": list[str],
  }
  ```

  定义 canonical_ir dict（顶层）：
  ```python
  {
    "version": str,
    "source_path": str,
    "nodes": dict,    # id → ir_node
    "scopes": dict,   # id → ir_scope
    "edges": list,    # [{"from": id, "to": id, "kind": str}]
    "source_map": dict,  # node_id → source_ref
    "meta": dict,
  }
  ```

  safe_level（写入 node meta）：
  ```python
  SAFE_LEVEL = {
    "L1": "implicit_to_explicit",   # 完全安全：desugaring / scope explicitization
    "L2": "trace_preserving",       # 条件安全：结构重组但 trace 不变
    "L3": "forbidden",              # 禁止：runtime-dependent（eval/reflection/monkey patch）
    "OPAQUE": "opaque",             # 系统无法分析，标记为黑盒
  }
  ```

  提供函数：
  ```python
  def empty_ir(source_path: str, version: str = "0.1") -> dict
  def make_ir_node(node_id: str, kind: str, label: str, scope: str, **kwargs) -> dict
  def make_scope(scope_id: str, kind: str, parent: str | None = None) -> dict
  def make_edge(from_id: str, to_id: str, kind: str = "data") -> dict
  ```

  kind 合法值（ir_node）：
  function / call / assign / branch / loop / return / side_effect / opaque

  kind 合法值（ir_scope）：
  module / function / class / block

  kind 合法值（edge）：
  data / control / scope / mutation

验证：
```
python -c "
import sys; sys.path.insert(0,'project')
from compiler.ir.schema import empty_ir, make_ir_node, make_scope, make_edge
ir = empty_ir('test.py')
n = make_ir_node('n1','function','add','scope_module')
s = make_scope('scope_module','module')
e = make_edge('n1','n2','data')
assert ir['version'] == '0.1'
assert n['kind'] == 'function'
assert s['kind'] == 'module'
assert e['kind'] == 'data'
print('OK')
"
```

## TASK-005
状态：已完成
做什么：写 Python AST → canonical IR 的 canonicalization
输入：TASK-004 的 expand result，TASK-002 的 ir/schema.py
输出：project/compiler/canon/python_canon.py

要求：
  函数签名：def canonicalize(expanded: dict) -> dict
  返回：canonical_ir dict（TASK-002 定义的格式）

  必须处理的 AST 节点（优先级从高到低）：

  FunctionDef / AsyncFunctionDef
    → ir_node kind="function"
    → 创建对应 scope（kind="function"）
    → source_ref 记录 lineno / end_lineno

  Call
    → ir_node kind="call"
    → label = 被调用函数名（静态可知时）或 "dynamic_call"（静态不可知时）
    → 如果是 print / open / requests.* / subprocess.* → meta["side_effect"] = True

  Assign / AugAssign / AnnAssign
    → ir_node kind="assign"
    → AugAssign（a += 1）必须 desugar 为 a = a + 1 的显式形式
      meta["desugared"] = True, meta["safe_level"] = "L1"

  If
    → ir_node kind="branch"
    → meta["has_else"] = bool

  For / While
    → ir_node kind="loop"
    → meta["loop_kind"] = "for" | "while"

  Return
    → ir_node kind="return"

  eval() / exec() / getattr() 动态调用
    → ir_node kind="opaque"
    → meta["safe_level"] = "L3"
    → meta["reason"] = "runtime-dependent"

  scope 规则：
    - module 级别：scope_id = "scope:module"
    - 每个 FunctionDef：scope_id = "scope:func:{name}:{lineno}"
    - 嵌套 scope 的 parent 正确指向外层 scope

  edge 规则：
    - Call 的 caller → callee：kind="control"
    - Assign 的 value → target：kind="data"
    - If 的 condition → branch node：kind="control"
    - 函数内节点 → 函数 scope：kind="scope"

验证：
```
python -c "
import sys; sys.path.insert(0,'project')
from pathlib import Path
from compiler.ingest.python_ingest import ingest
from compiler.macro.expand import expand
from compiler.canon.python_canon import canonicalize

    a = x + 1
    if a:

'''.strip(), encoding='utf-8')

assert 'assign' in kinds
assert 'branch' in kinds
```

## TASK-010
状态：已完成
做什么：写 structural pattern matcher
输入：canonical_ir dict，pattern list
输出：project/compiler/pattern/match.py

要求：
  函数签名：def match_patterns(ir: dict, patterns: list[dict]) -> list[dict]

  返回 match list：
  ```python
  {
    "pattern_id": str,
    "pattern_name": str,
    "node_ids": list[str],
    "scope_id": str,         # 哪个 scope 里匹配到的
    "confidence": float,     # v0 只用 1.0（确定性匹配）或 0.0（不匹配）
    "reason": str,
  }
  ```

  匹配规则（确定性，不用 LLM）：
  - 对每个 scope 里的 node 集合，检查是否满足 pattern 的 required_node_kinds
  - required_edge_kinds：检查该 scope 里的 edges 是否包含
  - min_nodes / max_nodes：scope 里的 node 数量是否在范围内
  - 所有条件满足 → confidence = 1.0
  - 不满足 → 不返回该 match

  不做近似匹配，不用 embedding，不用 LLM。

验证：
```
python -c "

def process(data):
    result = fetch(data)
    clean = transform(result)
    cache[data] = clean
    return clean
'''.strip(), encoding='utf-8')

  ```

## TASK-013
状态：已完成
做什么：写 canonical IR → graph projector
